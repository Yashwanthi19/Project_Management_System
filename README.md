# ProjeX website

## Features
- Project management
- Team collaboration
- Community building
- Time tracking

## Benefits
- Increased productivity
- Improved communication
- Better project outcomes
- Increased collaboration
- Networking opportunities

- NodeJs
- Express Js
- Mongo Db
- Styled Components
- Material-UI
- Nodemailer
